# **App Name**: WebGen

## Core Features:

- Admin Module: Allows administrators to manage doctors and patients.
- Doctor Module: Allows doctors to view and approve patient appointments.
- Patient Module: Allows patients to log in, manage their profile, and request appointments.
- AI Chatbot: Chat with AI using Gemini API regarding diseases; the AI acts as a medical advisor after receiving patient medical details from their profile and write a fallback thing to mention to consult doctor in critical things.
- Patient Medical Records: Stores patient's medical records in their profile.
- Appointment Request: Patients can request appointments with doctors.
- Appointment Approval: Doctors can approve or deny patient appointment requests.
- User Authentication: Secure login for admin, doctors and patients.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), symbolizing trust and reliability, suitable for the medical context.
- Background color: Light gray (#F0F4F8), for a clean, professional, and calming backdrop.
- Accent color: Teal (#26A69A) for CTAs and interactive elements, to convey a sense of health and vitality.
- Body and headline font: 'PT Sans', a humanist sans-serif that provides a modern, clear and easily readable text.
- Use clear, minimalist icons to represent different functionalities such as appointment types and medical departments.
- A clean, intuitive layout with clear separation of content sections. Use of white space to improve readability and reduce clutter.
